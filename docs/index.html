Installation

Dotnet global tool

If you have dotnet sdk install, you can install clizer as a dotnet global tool. Just run:
$ dotnet tool install -g clizer.dev
in your terminal of choice. Then you should be able to run the clizer tool :)

Download

You can download the clizer.exe below here. Then you need add it to enviroment path, like you would with java and other tools you use in your terminal.
